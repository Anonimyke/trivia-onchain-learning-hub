# Quiz Royale

## Setup

Since I cannot reliably run `npm install` in this environment, please run the following in your terminal:

```bash
cd trivia-app
npm install
npm run dev
```

## Features

- **Practice Mode**: Play casual trivia in 3 categories (Culture, Crypto, Sports).
- **Daily Streak**: Claim rewards daily. Streak resets if you miss a day.
- **Local Persistence**: Progress is saved in your browser.
- **Feedback System**: Dynamic messages based on your performance.

## Tech Stack

- Next.js 14 (App Router)
- React
- TypeScript
- CSS Modules (Global styles)

This project is a standalone repo as requested.
